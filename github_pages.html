Comment héberger un site web sur Github Pages.

1. Créer le projet sur github

Aller sur son compt github et créer un nouveau projet en cliquant sur new repository.

Créer un nouveau fichier nomé index.html et mettre un peu d'HTML dedans.

Cliquer en bas sur "commit this file" pour sauvegarder le fichier.

2. Configurer github pages

En haut à droite, cliquer sur "Settings", puis dans le menu de gauche, vers le bas, cliquer sur "Pages".

Choisir la branche "main" et sauvegarder en cliquant sur "Save".

Votre projet, a maintenant deux adresse : l'adresse des fichiers sources, et l'adresse du site, consultable sur le web.

Exemple pour ce site :
les fichiers source : https://github.com/MatysLSY/webdev
le site consultable : https://matyslsy.github.io/webdev/

Importer le site en local (sur son PC)

Pré-requis : il faut avoir installer GIT sur son PC.

Créer un dossier où l'on veut mettre tous ses projets.

Cloner le projet (repository) en local avec : git clone https://github.com/MatysLSY/webdev.git

On peut maintenant ouvrir et éditer le projet avec VScode.

Envoyer les modifs

On peut le faire avec le terminal : tous les fichiers modifiés ou créés, sont à ajouter avec git add.

Une fois qu'on a ajouté ces fichiers à enregistrer, on sauvegarde les modifications avec git commit -m "petit message explicatif"

Et enfin, on envoie les modifications sur github, avec git push origin main.
